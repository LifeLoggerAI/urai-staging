## Commands Required for Final Verification

Run from a fresh checkout:

```bash
git clone https://github.com/LifeLoggerAI/urai-staging.git
cd urai-staging
git checkout release/urai-staging-v1-integration-audit
npm --prefix functions ci
npm run check:deploy
npm run lint
npm run typecheck
npm run build
npm run test:unit
npm run test:e2e
```

`npm run test:e2e` requires Java/Firebase emulator support. In Firebase Studio/IDX, rebuild the Nix environment first if Java is missing.

## Deployment Verification Commands

```bash
firebase login
firebase use urai-staging
npm run deploy:staging
npm run smoke:staging
```

The deploy script performs install, readiness, lint, typecheck, build, unit tests, optional emulator tests, deploy, smoke tests, and writes `URAI_STAGING_LOCK.md` on success.

## Acceptance Criteria

Final lock requires all of the following evidence:

1. Install output from `npm --prefix functions ci`.
2. Readiness output from `npm run check:deploy`.
3. Lint/typecheck/build output.
4. Unit test output.
5. Emulator/rules test output.
6. Firebase deploy output proving deployment to `urai-staging` only.
7. Smoke output proving all HTTP checks pass.
8. Committed `URAI_STAGING_LOCK.md` generated by the deploy script.
